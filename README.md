# Todo List App

A simple, elegant, and fully functional todo list application built with HTML, CSS, and JavaScript.

## Features

- ✅ **Add Tasks** - Quickly add new tasks to your list
- ☑️ **Mark Complete** - Check off tasks as you complete them
- 🗑️ **Delete Tasks** - Remove tasks you no longer need
- 📊 **Statistics** - Track total, completed, and remaining tasks at a glance
- 💾 **Local Storage** - Your tasks are automatically saved to your browser's local storage
- 📱 **Responsive Design** - Works seamlessly on desktop and mobile devices
- 🎨 **Modern UI** - Beautiful gradient design with smooth animations
- 🧭 **Navigation Bar** - Clean navbar with app branding and navigation links

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)

### Installation

1. Clone or download this repository
2. Navigate to the project folder
3. Open `index.html` in your web browser

That's it! No installation or dependencies required.

## How to Use

1. **Add a Task**: 
   - Type your task in the input field
   - Click the "Add" button or press Enter

2. **Complete a Task**: 
   - Click the checkbox next to a task to mark it as completed
   - Completed tasks will appear with strikethrough text

3. **Delete a Task**: 
   - Click the "Delete" button next to the task you want to remove

4. **View Statistics**: 
   - Check the stats section at the bottom to see:
     - Total number of tasks
     - Number of completed tasks
     - Number of remaining tasks

## Features in Detail

### Local Storage
Your tasks are automatically saved to your browser's local storage. This means:
- Your tasks persist even after closing the browser
- No account or login required
- Data is stored locally on your device

### Empty State
When you have no tasks, a friendly message appears: "No tasks yet. Add one to get started!"

### Task Validation
- Empty tasks cannot be added
- Tasks are trimmed of whitespace before being saved
